BIO、NIO和AIO详细实例
============
###主要目标： 
 - 使用BIO、NIO和AIO造一个简单的轮子，提高对它们的理解
 - 深入了解BIO、NIO和AIO的底层实现
 - 能以此为基础，去分析各种框架，例如thrift、dubbo和netty等。
 
###代码概要： 
 - client，BIO、NIO和AIO客户端实现
 - common，客户端&服务端抽出来的公用代码
 - exception，自定义异常类
 - log，自定义日志
 - protocol，就是规定如何发送接收数据的，专业点叫做编解码
 - server，BIO、NIO和AIO服务端实现
